ETロボコン2010用NXT走行体(NXTway-ET) Java プログラム

					             2010/05/06 銀河系はねうま by Jacom Co.,LTD.
=================================================================================

■はじめに
本プログラムはETロボコン2010のNXT走行体(NXTway-ET)のJavaプログラムです。


■ファイル構成
倒立振子制御ライブラリと、サンプルプログラムが格納されています。

nxtway_et/
   |---src/
       |---etrobo/
           |---nxj/        ・・・倒立振子制御ライブラリ
           |---sample2010/ ・・・サンプルプログラム


■動作環境
本プログラムは以下の動作環境で確認を行っています。
・Windows XP Professional
・leJOS NXJ 0.85
・Eclipse 3.5 Galileo Pleiades All in One Java
・Version 0.85 of leJOS NXJ Eclipse Plugin

インストール方法については別紙、または leJOS 公式サイト
http://lejos.sourceforge.net/ を参照下さい。


■ビルド方法
- eclipse のワークスペースに、nxtway_et_2010 フォルダをインポートして下さい。
- 外部JARとして、leJOS NXJ インストールフォルダの lib/classes.jar を設定してください。
- Sample1Main.java を選択し、
  "Upload program to the NXT brick" でNXTにアップロードして下さい。

※その他のビルド方法については、leJOS 公式サイト を参照下さい。
  http://lejos.sourceforge.net/nxt/nxj/tutorial/Preliminaries/CompileAndRun.htm


■サンプルプログラムの操作方法
- NXTway-ETを立たせた状態で、反時計回りに黒線内側エッジ上、
  または時計周りに黒線外側エッジ上に配置します。
  この時、左右車輪は必ず地面に接地させてください。
- NXTway-ETの右側に装着されているタッチセンサを押すことで、ライントレースを開始します。
- LEFTボタン(左)によりロボットが停止します。
- ESCAPEボタン(下)によりプログラムが終了し、メニュー画面に戻ります。
- 万が一ボタンが効かなくなった場合は、リセットしてください。


■注意事項
- 本来バランス制御実行周期は 0.00400000019(秒)固定ですが、
  leJOS NXJ は nxtOSEK に比べて処理速度が遅いため、ユーザプログラムの内容により、
  実行周期がずれて、倒立制御に失敗することがあります。
  スリープ時間の変更、またはバランス制御実行周期パラメータの変更により、調節して下さい。

- なお、LEFTボタン(左)によりロボットが停止した際、バランス制御の実行回数と、
  走行開始から停止までの時間(ミリ秒)が表示されますので、参考にして下さい。
